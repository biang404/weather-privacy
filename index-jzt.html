一、我们如何收集和使用您的个人信息

我们仅在实现应用功能所必需的范围内处理信息，主要来源为：您主动输入、应用运行中产生的信息、以及第三方服务在您授权范围内提供的信息。

1.1 天气查询与展示（核心功能）

  • 信息类型：您主动输入的城市名称；根据该城市返回的天气数据（温度、湿度、风速等）。
  • 收集时机：您主动搜索城市、刷新天气时（前台）。
  • 处理目的：向天气服务接口查询并展示实时天气、小时预报、未来预报。
  • 不提供后果：将无法完成天气查询。

1.2 本地偏好与历史记录

  • 信息类型：搜索历史、收藏城市、主题模式、字体缩放、动画开关、温度单位、隐私同意状态。
  • 收集时机：您使用相关功能时写入本地。
  • 处理目的：保存您的个性化设置，提升使用体验。
  • 不提供后果：将无法保存个性化设置或历史记录。

1.3 广告展示（京媒 SDK）

本应用集成京媒（京东）广告 SDK，用于展示开屏广告。依据京媒公开文档，其可能处理以下信息：

  • 必选信息：设备制造商、设备型号、设备品牌、系统版本、网络状态、应用包名/版本号、广告展示/点击等交互数据、性能与崩溃数据、传感器信息（如加速度/旋转矢量）。
  • 可选信息（基于授权）：地理位置信息、IP 地址、设备标识符（如 OAID）。
  • 收集时机：在您同意隐私政策后初始化广告 SDK 时。
  • 处理目的：广告投放、广告监测归因、反作弊与稳定性保障。
  • 不提供后果：可能影响广告展示效果，但不影响天气查询核心功能。

1.4 敏感个人信息说明

我们不会主动采集身份证、护照、银行卡、人脸、指纹、通讯录、短信内容等信息。对于可能涉及敏感属性的信息（如精确位置信息），仅在您授权并用于对应场景时处理，您可随时通过系统权限设置关闭。

1.5 个性化广告关闭方式

京媒 SDK 在获得位置等可选授权时可能用于广告优化。您可通过以下方式关闭相关个性化能力：

  • 在系统中关闭本应用的位置权限（路径示例：系统设置 → 应用管理 → 讯天气 → 权限）。
  • 在系统广告隐私设置中关闭广告个性化/重置广告标识符（具体路径因设备品牌与系统版本不同而异）。

二、权限调用说明

我们在 Android 端声明了以下权限。不同权限对应不同功能，您可在系统设置中管理授权。

权限 用途说明 是否必要
INTERNET 访问天气接口、广告接口 必要
ACCESS_NETWORK_STATE 判断网络状态，优化请求与广告加载 必要
ACCESS_WIFI_STATE 判断网络连接状态，提升请求稳定性 可选
ACCESS_COARSE_LOCATION / ACCESS_FINE_LOCATION 在授权情况下用于广告投放优化与反作弊 可选
READ_PHONE_STATE 在授权情况下用于设备状态识别与广告反作弊 可选
READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE 在兼容版本上用于广告素材缓存与读取 可选
QUERY_ALL_PACKAGES 用于广告相关能力兼容与风险识别（如反作弊场景） 可选

三、我们如何使用 Cookie 及其他同类技术

本应用为原生移动应用，不依赖浏览器 Cookie 提供核心能力。我们会使用本地存储（如 SharedPreferences）保存必要设置与历史记录，用于实现本地功能，不用于跨应用跟踪。

四、我们如何委托处理、共享、转让、公开披露您的个人信息

4.1 委托处理

为实现天气查询、广告展示等功能，我们会接入第三方服务。我们会尽商业上合理努力要求合作方按照法律法规处理个人信息。

4.2 第三方共享明细

第三方主体/服务 共享或处理信息类型 目的/场景 处理方式 隐私政策链接
京媒 SDK（京东及其关联公司） 设备信息、应用信息、广告交互数据、性能数据；可选的位置、IP、OAID 开屏广告展示、归因、反作弊、稳定性优化 SDK 本机采集并网络传输 https://opendoc.jd.com/janGroup/help/agreement/privacy.html

4.3 转让

我们不会将您的个人信息转让给任何公司、组织或个人；如未来发生合并、收购、资产转让等情形，我们将依法履行告知义务。

4.4 公开披露

我们原则上不会公开披露您的个人信息，仅在取得您明确同意或法律法规要求的情形下进行披露。

五、我们如何保护您的个人信息

我们采取合理可行的安全措施保护信息安全，包括但不限于：访问权限控制、传输安全措施、日志审计、异常监控与应急响应机制。请您理解，互联网并非绝对安全环境，我们将持续提升安全能力。

六、我们如何存储您的个人信息

6.1 存储地点

  • 本地数据（搜索历史、收藏、设置等）主要存储于您的设备本地。
  • 广告相关与天气接口数据由相应第三方服务商处理并存储，详见其隐私政策。

6.2 跨境传输说明

由于 OpenWeather 服务由境外主体提供，您发起天气查询时，相关请求数据可能传输至境外服务器。境外接收方信息如下：

  • 接收方：OpenWeather Ltd
  • 目的：提供天气查询服务
  • 类型：城市查询信息、接口请求相关信息
  • 联系方式（公开渠道）：info@openweathermap.org

6.3 存储期限

  • 搜索历史：保留最近10条，您可在搜索页手动清空。
  • 收藏城市与主题设置：保留至您主动删除、清除应用数据或卸载应用。
  • 隐私同意状态：保留至您清除应用数据或卸载应用。

七、您的权利

您对个人信息享有访问、更正、删除、复制、撤回同意等权利。您可通过以下方式行使：

  • 访问/更正：在应用内调整主题、字体、历史、收藏等本地数据。
  • 删除：在搜索页清空历史，或通过系统卸载应用/清除应用数据。
  • 撤回同意：可通过系统权限管理关闭对应权限，或停止使用本应用。
  • 联系申诉：通过本政策第十一部分的联系方式提交请求。

为保障安全，我们可能在受理前核验身份。我们通常会在 15个工作日 内回复您的请求。

八、我们如何处理未成年人的个人信息

我们重视未成年人信息保护。本应用不以未成年人为主要服务对象。若您是未成年人，请在监护人指导下使用；如监护人发现未成年人信息被不当处理，可通过本政策联系方式与我们联系。
